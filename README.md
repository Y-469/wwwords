# WordDrill - 英単語トレーニングアプリ

## 概要

GTEC対策を目的とした英語学習Webアプリ。単語ドリル・ライティング・読解クイズの3分野を1つのHTMLファイル (`index.html`) で提供する。GitHub Pages でホスティングし、Firebase Realtime Database によるクラウド同期に対応。

### 主な機能

| 機能 | 説明 |
|------|------|
| **単語クイズ** | 英→日の4択クイズ（94語） |
| **ライティング** | 日本語の意味から英単語をタイピングで回答 |
| **推論読解** | 150語程度の英文パッセージ＋4択問題（300問、6タイプ） |
| **空欄補充** | 文中の空欄に適切な語句を選ぶ4択問題（300問） |
| **長文読解** | 長文パッセージに対する複数設問形式（20パッセージ） |
| **未知単語リスト** | 読解中にマークした単語を一覧管理 |
| **ダッシュボード** | 単語別の正答率・学習履歴の確認 |
| **クラウド同期** | Firebase経由で複数デバイス間のデータ同期 |
| **セッション中断/再開** | 途中で閉じても続きから再開可能 |

### 技術スタック

- HTML / CSS / JavaScript（単一ファイル構成、約3,700行）
- ダークテーマUI（CSS変数）
- localStorage によるデータ永続化
- Firebase Realtime Database（オプション）
- GitHub Actions による自動デプロイ
